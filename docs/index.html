Формування геометричних параметрів класифікатора

Як інформаційна міра використано ентропійну міру Кульбака

В процесі формування вирішувальних правил всі класи розпізнавання сформували контейнери класів

  1. Проаналізовано сучасні тенденції розвитку класифікаторів.
  2. Визначено переваги гібридної архітектури з нейромережевим екстрактором ознак і лінійною класифікацією в рамках геометричного підходу до формування вирішувальних правил.
  3. Спроектовано архітектуру моделі машинного навчання гібридної моделі розпізнавання типів місцевості за фотоматеріалами.
  4. Програмно реалізовано алгоритм машинного навчання системи мовою програмування Python на основі технології tensorflow. 
  5. Проаналізовано результати практичного використання системи для розпізнавання місцевості в режимі тестової класифікації, що продемонструвало точність класифікації у 94%.

Гібридна модель машинного навчання системи розпізнавання місцевості за матеріалами фотозйомки з нейромережевим екстрактором ознак і інформаційно-екстремальним інтелектуальним класифікатором
